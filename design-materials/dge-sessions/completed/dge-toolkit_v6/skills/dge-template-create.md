# Skill: DGE テンプレート作成

## Trigger
ユーザーが以下のいずれかを言ったとき:
- 「DGE テンプレートを作りたい」
- 「新しい DGE パターンを作って」

## 手順

### Step 1: テーマを聞く
「どんなテーマの DGE テンプレートですか？」

### Step 2: Scene を設計
テーマに基づいて 3-5 の Scene を提案:
「このテーマでは以下の Scene を提案します:
  Scene 1: ...
  Scene 2: ...
  修正があれば教えてください。」

### Step 3: キャラクターを選定
characters/catalog.md を参照して推奨キャラを提案。
ユーザーが変更・追加できる。

### Step 4: テンプレートを生成
以下の形式で dge-method/templates/ にファイルを作成:

```markdown
# DGE Template: [テーマ]

## 概要
[1 行の説明]

## 推奨キャラクター
[キャラ名] + [キャラ名] + [キャラ名]

## Scene 構成

### Scene N: [タイトル]
\```
focus: [このシーンの焦点]
[キャラ1]: 「[典型的なセリフ]」
[キャラ2]: 「[典型的なセリフ]」

抽出すべき Gap:
- [gap 1]
- [gap 2]
\```
```

### Step 5: テスト実行
「テンプレートを 1 回実行してテストしますか？」
→ Yes なら dge-session.md の skill で実行
→ 結果を見てテンプレートを調整
