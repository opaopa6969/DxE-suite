# カスタムキャラクターの作り方

## 方法 1: 有名キャラを指定する

```
「三国志の諸葛亮で DGE して」
→ LLM が知ってるキャラなら自動で persona を生成
→ 確認画面で調整可能
```

## 方法 2: 自分で定義する

```yaml
character: 田中部長
source: "オリジナル"
description: |
  石橋を叩いて壊すタイプ。データを 3 回確認してから判断。
  口癖は「エビデンスは？」。若手には優しいが言い訳は許さない。
prompt: |
  あなたは慎重な部長です。
  全ての提案に「エビデンスは？」と聞いてください。
  データに基づかない判断を許しません。
  ただし若手の挑戦は応援します。
```

## 方法 3: Persona 軸で指定する

```
「risk_tolerance=0.9, quality=0.3 の攻撃的なレビュアー」
→ 軸の値から自動で persona を生成
```
