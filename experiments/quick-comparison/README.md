# DGE vs 素の LLM — クイック比較実験

## 目的

「DGE は素の LLM に頼むより良い結果を出すか？」を簡易的に検証する。

## 実験設計

### 条件
- **A（素の LLM）**: 「この設計をレビューして。問題点を全部挙げて」
- **B（DGE）**: 「この設計を DGE して」（キャラ + パターン + Scene 構造）

### テーマ（3 つ）
1. 認証 API の設計
2. 課金モデルの設計
3. レガシーコードのリファクタ計画

### 手順
各テーマに対して:
1. A（素）を実行 → `results/theme-N-plain.md` に保存
2. B（DGE）を実行 → `results/theme-N-dge.md` に保存
3. 人間が比較して `results/theme-N-comparison.md` に記録

### 評価指標
| 指標 | 測り方 |
|------|--------|
| Gap 数 | 単純カウント |
| 非自明な Gap の数 | 人間が「これは気づかなかった」と判定した数 |
| Severity 分布 | Critical / High / Medium / Low の比率 |
| カバレッジ | カテゴリ（11 種）のうち何種に Gap があるか |
| 会話劇による追加発見 | DGE にのみ存在して A にない Gap の数 |
